# 📝 系统要求

## 基本要求
- **Python 3.7+** （推荐 3.8+）
- **网络连接** （访问 EasyEDA/LCSC API）
- **KiCad 5.x 或 6.x+** （使用生成的库文件）

## Python 依赖
- **Flask 2.0+** （Web UI）
- **Flask-CORS** （跨域支持）
- **requests** （HTTP 请求）
- **其他依赖** 见 requirements.txt

## 支持的操作系统
- ✅ Windows 10/11
- ✅ macOS 10.14+
- ✅ Linux (Ubuntu 18.04+)

## 🎯 支持的元件类型

- 🔌 **连接器** - 各种插头和端子
- 🔧 **分立元件** - 电阻、电容、电感、二极管等
- 💾 **集成电路** - MCU、存储器、运放等
- ⚡ **电源管理** - 稳压器、开关电源芯片等
- 📡 **RF元件** - 天线、滤波器等
- 🔍 **传感器** - 温度、压力、光学传感器等
